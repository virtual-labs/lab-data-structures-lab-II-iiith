* Steps to build and run a lab
  *STEP 1 :* Clone the lab repository
  #+BEGIN_EXAMPLE
  git clone https://github.com/vitual-labs/data-structures-lab-I.git
  #+END_EXAMPLE

  *STEP 2 :* Change directory to lab repo
  #+BEGIN_EXAMPLE
  cd data-structures-lab-I
  #+END_EXAMPLE

  *STEP 3 :* Removes the build directory and builds the sources
  #+BEGIN_EXAMPLE
  make -k all
  #+END_EXAMPLE
  
  *STEP 4 :* Run the lab server
  #+BEGIN_EXAMPLE
  make -k run
  #+END_EXAMPLE
